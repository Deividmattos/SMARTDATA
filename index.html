Projetos - acompanhemento de notas fiscais ERP

modelados dentro do Power BI para criar dashboards

Dash para acompanhamento das notas fiscais emitidas e recebidas no sistema ERP. O objetivo era criar uma solução que permitisse ao time financeiro monitorar em tempo real todas as operações fiscais, garantindo que as informações estivessem sempre atualizadas e acessíveis para a tomada de decisões.

Integração direta com o ERP para extrair os dados de notas fiscais, incluindo informações como data de emissão, valor, cliente ou fornecedor, status de pagamento e impostos. Esses dados foram transformados e modelados dentro do Power BI para criar dashboards interativos, onde os usuários podem visualizar tendências, identificar notas em aberto ou com problemas, e fazer análises detalhadas de acordo com diferentes filtros e critérios.

Projetos - Produtividade operacional

Dash de produtividade que acompanham a quantidade de tarefas fechadas diariamente, permitindo uma análise detalhada da eficiência operacional. O objetivo principal era fornecer ao time de gestão uma visão clara do desempenho das equipes e identificar oportunidades para melhorar a produtividade.

Os dados de tarefas concluídas foram extraídos e integrados ao Power BI, onde foram modelados para gerar relatórios que mostram a quantidade de tarefas fechadas dia a dia. Além disso, implementei cálculos específicos para determinar a produtividade diária com base nos dias úteis, ajustando os resultados para refletir corretamente o desempenho em períodos com feriados ou fins de semana. Com os dashboards interativos criados, os gestores podem filtrar os dados por diferentes períodos, comparar a produtividade entre dias, semanas ou meses, e identificar padrões ou desvios que necessitam de intervenção.

Projetos - Banco de horas - RH

Relatório de acompanhamento de banco de horas, que permite monitorar de forma eficiente o saldo de horas trabalhadas pelos funcionários. O relatório oferece uma visão detalhada, mostrando tanto as horas positivas quanto as negativas para cada funcionário, segmentadas por regional. Essa ferramenta é fundamental para o controle de jornada, auxiliando na tomada de decisões e garantindo conformidade com as políticas de trabalho da empresa.
